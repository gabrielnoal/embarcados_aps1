# Resumo :

Esse exemplo demonstra o controle do módulo de `OLED 1 XPLAINED PRO`  via interface SPI (Serial Peripheral Interface Bus).

Periféricos uC:

    - Power Managment Controller (PMC)
    - Serial Protocol Interface (SPI)
 
APIs :

    - Driver SSD1306
 
Módulos : 

    - OLED 1 XPLAINED PRO

## Diagrama

#OLED :
#    - LED1 : PA0
#    - LED2 : PC30
#    - LED3 : PB2
#    - BUT1 : PD28
#    - BUT2 : PC31
#    - BUT3 : PA19
BUZZER :
    + : PC13
    - : GND
